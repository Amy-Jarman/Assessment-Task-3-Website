Name Web Attacks
Malware

Image source: here 		Malware (malicious software) is any program or code that is created with the intent to do harm to a computer, network or server. There are many different types of malware including trojans, spyware, ransomware, viruses, worms, adware, and rootkits to name a few.
More info here
Denial of Service (DoS)

Image source: here 		A Denial-of-Service (DoS) attack floods a network with false requests in order to disrupt business operations. During a DoS attack, users are unable to perform routine tasks such as accessing email, websites, accounts etc. DoS attacks cost organisations time and money in order to restore critical business operations.
More info here
Man-in-the-Middle (MITM) Attacks

Image source: here 		Man-in-the-middle attacks are where an attacker gains access to eavesdrop and spy on data sent back and forth between two people, networks or computers. The attacker can modify or access the message before it reaches its destination. Using strong encryption on access points or VPNs can help mitigate this type of attack.
More info here
SQL Injection

Image source: here 		A Structured Query Language (SQL) injection is a common method of taking advantage of websites that depend on databases to serve their users. It uses a SQL query sent from the client to a database on the server. A command is inserted, or “injected”, into a data plane in place of a password or login, the server runs the command and the system is penetrated.
More info here
Cross-Site Scripting

Image source: here 		Cross-Site Scripting (XSS) attacks are a type of injection, in which malicious scripts are injected into otherwise benign and trusted websites. XSS attacks occur when an attacker uses a web application to send malicious code, generally in the form of a browser side script, to a different end user. An attacker can use XSS to send a malicious script to an unsuspecting user.
More info here
Phishing

Image source: here 		Phishing uses email, SMS, phone, social media, and social engineering techniques to entice a victim to share sensitive information (such as passwords, account numbers etc) or to download a malicious file that will install viruses on their computer or phone.
More info here
DNS Spoofing

Image source: here 		Domain Name System (DNS) spoofing is where a hacker alters DNS records to send traffic to a fake or “spoofed” website. Once on the fraudulent site, the victim may enter sensitive information that can be used or sold by the hacker.
More info here
Ransomware

Image source: here 		Ransomware is a form of malware. Cybercriminals use ransomware as a tool to steal data and essentially hold it hostage. They only release the data when they receive a ransom payment. Organizations most vulnerable to ransomware attacks hold sensitive data, such as personal information, financial data, and intellectual property.
More info here
Drive-by Download Attacks

Image source: here 		Drive-by download attacks refer to malicious programs that install to devices without the users consent. This also includes unintentional downloads of any files or bundled software onto computer devices. Masked in all corners of the web, these attacks cause even perfectly legitimate sites to spread this threat.
More info here
Macro Viruses

Image source: here 		A macro virus is a computer virus written in the same macro language used to create software programs such as Microsoft Excel or Word. It centers on software applications and does not depend on the operating system (OS). As a result, it can infect any computer running any kind of OS, including Windows, macOS and Linux.
